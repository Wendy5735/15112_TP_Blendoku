Game name: Blendoku

Description: The player is given a randomly shaped grid of squares, some of which 
are filled with colors and others are empty. A set of additional color squares is 
also provided in the waiting zone. To clear a level, the player must fill the empty 
squares in the board using the color squares in the waiting zone in order to create 
smooth color sequences (gradients) both horizontally and vertically.

Instruction: Run the file "Blendoku.py"

No libraries need to be installed. No shortcut commands.
